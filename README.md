> ## 功能介绍
> 本系统为实现疫苗预约而打造的“疫苗在线预约系统”，包括患者、医生、管理员三个模块
>
> 
> :star:患者功能概括为：登录注册、疫苗预约、医师预约、
>
> :star:医生功能概括为：登录注册、查看预约信息、发放疫苗、住院申请等
>
> :star:管理员功能概括为：登录、患者管理、医生管理、疫苗管理、疫苗预约管理、住院管理、排班管理、数据统计与分析等
